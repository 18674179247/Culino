## 应用功能流程

### 登录流程
```
启动应用
  ↓
登录页面
  ↓
输入用户名密码
  ↓
登录成功
  ↓
进入主界面（带底部导航）
```

### 主界面导航
```
底部导航栏
├── 首页（菜谱列表）
│   ├── 显示随机推荐的菜谱
│   ├── 支持搜索和筛选
│   ├── 点击菜谱 → 菜谱详情
│   └── 支持分页加载
├── 收藏
│   ├── 显示收藏的菜谱
│   ├── 点击菜谱 → 菜谱详情
│   └── 取消收藏功能
└── 我的
    ├── 显示个人信息
    ├── 编辑资料
    └── 登出功能
```

### 菜谱详情页面
```
菜谱详情
├── 基本信息（标题、描述、难度、时间、人份）
├── 食材列表
├── 调料列表
├── 烹饪步骤
└── 营养信息（如果有）
```

## 测试步骤

1. **启动应用**
   ```bash
   ./gradlew :composeApp:run
   ```

2. **登录**
   - 输入用户名和密码
   - 点击登录

3. **浏览菜谱**
   - 登录后自动进入首页
   - 看到随机推荐的菜谱列表
   - 点击任意菜谱查看详情

4. **查看收藏**
   - 点击底部"收藏"标签
   - 查看已收藏的菜谱
   - 可以取消收藏

5. **个人中心**
   - 点击底部"我的"标签
   - 查看个人信息
   - 点击登出返回登录页
