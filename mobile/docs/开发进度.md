    )

composable(Routes.RECIPE_DETAIL) { backStackEntry ->
    val recipeId = backStackEntry.arguments?.getString("recipeId") ?: return@composable

        recipeId = recipeId,

}

## 技术说明

### 导航架构
```
App
├── 未登录状态
│   ├── 登录页面
│   └── 注册页面
└── 已登录状态
    └── MainScreen (带底部导航)
        ├── 首页 (菜谱列表)
        ├── 收藏
        └── 我的 (个人资料)
```

### 状态管理
- 使用 `remember { mutableStateOf(false) }` 管理登录状态
- 登录成功后 `isLoggedIn = true`
- 登出后 `isLoggedIn = false`
