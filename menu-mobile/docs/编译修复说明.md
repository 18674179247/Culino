# 编译错误修复说明

## 问题原因

新创建的 `feature:recipe` 和 `feature:social` 模块缺少必要的依赖和配置，导致编译失败。

### 主要错误

1. **缺少 kotlinx.serialization 依赖**
   - 错误：`Unresolved reference 'serialization'`
   - 原因：使用了 `@Serializable` 和 `@SerialName` 注解但没有添加序列化库

2. **缺少 ApiResponse 和 Result 类型**
   - 错误：`Unresolved reference 'ApiResponse'`、`Unresolved reference 'Result'`
   - 原因：这些类型定义在 `core:network` 和 `core:common` 模块中，但依赖配置不完整

3. **缺少 Koin 依赖**
   - 错误：`Unresolved reference 'koin'`、`Unresolved reference 'koinViewModel'`
   - 原因：UI 代码中使用了 Koin 依赖注入但没有添加依赖

4. **缺少 Android 配置**
   - 使用了 `jvm()` 而不是 `androidTarget()`
   - 缺少 `android {}` 配置块

## 修复方案

### 1. feature/recipe/build.gradle.kts

**添加的插件：**
```kotlin
alias(libs.plugins.androidLibrary)
alias(libs.plugins.kotlinSerialization)
```

**修改的配置：**
```kotlin
kotlin {
    androidTarget()  // 改为 androidTarget
    // ...
}
```

**添加的依赖：**
```kotlin
implementation(libs.kotlinx.serialization.json)
implementation(libs.koin.core)
implementation(libs.koin.compose)
```

**添加的 Android 配置：**
```kotlin
android {
    namespace = "com.menu.feature.recipe"
    compileSdk = libs.versions.android.compileSdk.get().toInt()
    defaultConfig {
        minSdk = libs.versions.android.minSdk.get().toInt()
    }
    compileOptions {
        sourceCompatibility = JavaVersion.VERSION_17
        targetCompatibility = JavaVersion.VERSION_17
    }
}
```

### 2. feature/social/build.gradle.kts

应用了相同的修复：
- 添加 `androidLibrary` 和 `kotlinSerialization` 插件
- 改为 `androidTarget()`
- 添加序列化和 Koin 依赖
- 添加 Android 配置块

## 依赖说明

### 必需的依赖

1. **kotlinx.serialization.json**
   - 用于 JSON 序列化/反序列化
   - 支持 `@Serializable` 和 `@SerialName` 注解

2. **koin.core + koin.compose**
   - 依赖注入框架
   - 支持 `koinViewModel()` 函数

3. **kotlinx.datetime**
   - 日期时间处理
   - 用于 `Instant` 类型

4. **androidx.lifecycle.viewmodel**
   - ViewModel 基类
   - 生命周期管理

### 已有的依赖（通过 core 模块）

- `core:common` - 提供 `Result` 类型
- `core:network` - 提供 `ApiClient` 和 `ApiResponse` 类型
- `core:ui` - 提供 UI 组件

## 验证步骤

修复后，执行以下命令验证：

```bash
# 清理构建
./gradlew clean

# 编译 recipe 模块
./gradlew :feature:recipe:build

# 编译 social 模块
./gradlew :feature:social:build

# 编译整个项目
./gradlew build
```

## 注意事项

1. **Gradle 权限问题**
   - 如果遇到 `Operation not permitted` 错误
   - 可能是 macOS 权限限制
   - 尝试在 Android Studio 中构建

2. **依赖版本**
   - 所有依赖版本由 `libs.versions.toml` 统一管理
   - 确保版本兼容性

3. **模块结构**
   - 所有 feature 模块应该有相同的基础配置
   - 包括：plugins、kotlin target、android 配置
