# 前端编译问题最终修复

## 问题根源

项目使用的是 **Kotlin Multiplatform** 架构，不是纯 Android 项目。因此：
1. ❌ 不应该使用 `androidTarget()` 和 `android {}` 配置
2. ❌ 不应该使用 `androidLibrary` 插件
3. ✅ 应该使用 `jvm()` target
4. ❌ 项目中没有配置 Koin 依赖注入

### 依赖注入方式

项目使用 **手动依赖注入**，通过 `AppComponent` 类：

```kotlin
class AppComponent(dataStorePath: String) {
    // 手动创建依赖
    private val apiClient by lazy { ApiClient(httpClient) }
    private val recipeApi: RecipeApi by lazy { RecipeApiImpl(apiClient) }
    
    // 提供 ViewModel 工厂方法
    fun recipeListViewModel() = RecipeListViewModel(...)
    fun recipeDetailViewModel() = RecipeDetailViewModel(...)
}
```

在 UI 中使用：
```kotlin
val viewModel = remember { appComponent.recipeListViewModel() }
RecipeListScreen(
    viewModel = viewModel,
    onRecipeClick = { ... }
)
```
